Detailed Description

Processes camera images to detect lane positions using GPU-accelerated OpenCV.

Subscribes to raw camera images, applies thresholding, morphological operations, and Hough line detection to identify lane markings. Publishes detected lane positions as lane_msgs/msg/LanePositions and optional debug images (edges/mask).

Applies GPU-accelerated preprocessing pipeline to an image.

Steps:

  1. Thresholding (adaptive based on is_white_lane parameter).
  2. Morphological transformations (dilation + erosion).
  3. ROI cropping (trapezoidal mask).
  4. Canny edge detection.
